# Plano Nutricional 💪

Dashboard interativo e mobile-first para acompanhamento diário do plano nutricional.

## Funcionalidades

- **5 refeições** organizadas em timeline vertical com cards colapsáveis
- **Tabela de substituições** com tiers coloridos (Sempre / Aceitável / Raramente)
- **Checkboxes diários** para marcar refeições feitas (reseta automaticamente a cada dia)
- **Barra de progresso** do dia
- **Links cruzados** entre refeições e substituições
- **Auto-expand** da refeição mais próxima do horário atual
- Otimizado para **iPhone** (safe areas, touch targets 44px)

## Como usar

Abra `plano-nutricional.html` no navegador — arquivo único, zero dependências externas.

## Stack

HTML + CSS + JS inline. Self-contained.
